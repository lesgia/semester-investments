# Semester Investments Project

## Introduction
This repository is for managing and tracking semester investments.

## Features
- Investment tracking and management.
- Performance analytics and reporting.

## Technology Stack
- Python
- Pandas for data analysis
- Matplotlib for visualizations

## Usage
Clone the repository and run the main script:
```bash
git clone https://github.com/lesgia/semester-investments.git
cd semester-investments
python main.py
```
